Listagem de tags HTML

  • <style>

    • contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, geralmente no CSS. Deve estar contido na head do documento html, mas também pode definir a cor específica de um parágrafo.

    Por exemplo, o texto está estilizado em roxo com essa tag, mas esse parágrafo está em azul.

  • <label>

    • representa uma legenda para um item em uma interface de usuário. Ele pode estar associado com um elemento de controle, colocando este dentro do elemento label, ou usando o atributo for. Tal controle é chamado o controle etiquetado do elemento etiqueta. Um input pode ser associado a diversas etiquetas (labels).

  • <input>

    • é usado para receber dados de um usuário, e é complexo pois tem diversos atributos, pode ser inserido um botão para click ou uma checkbox, por exemplo.

    Exemplo de input e label:

    Marque se você gosta de refrigerante.
    Marque se você gosta de suco.

  • <i>

    • representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico". Desse jeito.

  • <strong>

    • dá ao texto uma forte importância, e é tipicamente mostrado em negrito. O exemplo está nos títulos das seções de conteúdo.

  • <small>

    • representa comentários secundários em uma letra menor, é usado, por exemplo, em textos de copyright.
