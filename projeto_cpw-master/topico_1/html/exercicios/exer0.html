Listagem das principais tags da HTML:

exemplo de sessao

  • <head>

    • providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

  • <style>

    • contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, geralmente no CSS.

  • <title>

    • define o título do documento, mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas.

  • <body>

    • representa o conteúdo de um documento. É permitido apenas um <body> por documento.

  • <footer>

    • representa um rodapé para o seu sectioning content (conteúdo de seção) mais próximo ou sectioning root elemento (ou seja, seu parente mais próximo article, aside, nav, section, blockquote, body, details, fieldset, figure, td). Normalmente um rodapé contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.

  • <label>

    • representa uma legenda para um item em uma interface de usuário. Ele pode estar associado com um elemento de controle, colocando este dentro do elemento label, ou usando o atributo for. Tal controle é chamado o controle etiquetado do elemento etiqueta. Um input pode ser associado a diversas etiquetas (labels).

  • <i>

    • representa uma parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios. Normalmente, é apresentado com o uso do tipo "itálico".

  • <strong>

    • dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

  • <small>

    • representa comentários secundários em uma letra menor, é usado, por exemplo, em textos de copyright.

  • <br>

    • produz uma quebra de linha no texto

  • <hr>

    • representa uma quebra de elementos no nível de parágrafos, geralmente é usado em mudanças de tópico na mesma seção ou mudanças de cena.

  • <b>

    • é usado para atrair a atenção do leitor nos conteúdos desse elemento, os quais não seriam de uma importância especial. Era usado como o negrito, mas atualmente o negrito deve ser feito pelo tag <strong> para demonstrar algo importante, e a estilização deve ser feita em CSS.

  • <div>

    • é um elemento block-level usado genericamente para o fluxo de conteúdo , mas não tem efeito no conteúdo ou layout da página até ser estilizado com CSS.

  • <form>

    • representa uma seção de conteúdo interativo para submeter informações, como um formulário.

  • <input>

    • é usado para receber dados de um usuário, e é complexo pois tem diversos atributos, pode ser inserido um botão para click ou uma checkbox, por exemplo.

  • <tr>

    • define uma fileira(linha) de células em uma tabela

  • <td>

    • define uma das células da linha criada pelo comando anterior

  • <span>

    • é um elemento muito similar ao div mas inline level. serve como um container para o fluxo de conteúdo e pode ser estilizado por CSS. pode ser usado para agrupar elementos, por exemplo, por classe.

  • <img>

    • insere uma imagem.

  • <audio>

    • insere áudio, o qual pode ter uma ou mais de uma fonte, representada pelo atributo src ou pelo elemento <source>, também pode ser usado como mediastream.

  • <canvas>

    • usado para desenhar gráficos e animações.

  • <datalist>

    • representa uma série de opções para o usuário escolher e o sistema registrar.

  • <details>

    • criará um ícone que mostra detalhes ao aproximar o cursor.

  • <embed>

    • incorpora conteúdo externo, como de um plug-in do browser

  • <ul>

    • representa uma lista desordenada, geralmente com ícones.

  • <ol>

    • representa uma lista em ordem.

  • <li>

    • representa um item em uma lista.
