Guia de Tags HTML

Lista de Tags

  • <head>

    • Providencia informações gerais (metadados) sobre o documento(ex. charset, style), incluindo seu título e links para scripts e folhas de estilos.

  • <style>

    • Contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, geralmente no CSS.

    Além da tag style, existe o atributo style, que pode ser usado para estilizar um paragrafo, ou dentro da tag <span>, para estilizar algo em inline level.

  • <title>>

    • Título do documento, mostrado na barra de título de um navegador ou na aba da página. Pode conter somente texto e quaisquer marcações contidas no texto não são interpretadas.

  • <body>

    • Corpo/conteúdo de um documento. É permitido apenas um <body> por documento.

  • <footer>

    • Rodapé para o seu sectioning content (conteúdo de seção) mais próximo ou sectioning root elemento (ou seja, seu parente mais próximo article, aside, nav, section, blockquote, body, details, fieldset, figure, td). Normalmente um rodapé contém informações sobre o autor da seção de dados, direitos autorais ou links para documentos relacionados.

  • <label>

    • Representa uma legenda/etiqueta para um item em uma interface de usuário. Ele pode estar associado com um elemento de controle, colocando este dentro do elemento label, ou usando o atributo for. Tal controle é chamado o controle etiquetado do elemento label. Um input pode ser associado a diversas labels.

  • <i>

    • Parte do texto que é destacada do restante por algum motivo, por exemplo, termos técnicos, expressões de outros idiomas ou pensamentos de personagens fictícios.
    Normalmente, é apresentado com o uso do tipo "itálico".

  • <strong>

    • Dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

  • <small>

    • Comentários secundários em uma letra menor, é usado, por exemplo, em textos de copyright.

  • <br>

    • Produz uma quebra de linha no texto.

  • <hr>

    • Quebra de elementos no nível de parágrafo, geralmente é usado em mudanças de tópico na mesma seção ou mudanças de cena.

  • <b>

    • É usado para atrair a atenção do leitor nos conteúdos desse elemento, os quais não seriam de uma importância especial.

    Era usado como o negrito, mas atualmente o negrito deve ser feito pelo tag <strong> para demonstrar algo importante, e a estilização deve ser feita em CSS.

  • <div>

    • Elemento block-level usado genericamente para o fluxo de conteúdo , mas não tem efeito no conteúdo ou layout da página até ser estilizado com CSS.

  • <form>

    • Usado para que o user submeta informações: um formulário.

  • <input>

    • É usado para receber dados de um usuário, e é complexo pois tem diversos atributos, pode ser inserido um botão para click ou uma checkbox, por exemplo.

  • <tr>

    • Define uma fileira(linha, row) de células em uma tabela.

  • <td>

    • Define uma das células da linha criada pelo comando anterior.

  • <th>

    • Define uma das headers da tabela.

  • <span>

    • Muito similar ao <div> mas inline level. Serve como um container para o fluxo de conteúdo e pode ser estilizado por CSS. Pode ser usado para estilizar (com o atributo style) e para agrupar elementos, por exemplo, por classe (atributo class).

  • <img>

    • Insere uma imagem. Acompanhada do atributo "src". É aconselhável que o diretório seja "./" pois encaminhará para o Relative Path do usuário; Nesse caso a imagem deve estar contida na mesma pasta que o código fonte, ou em uma mais interna.

  • <audio>

    • Insere áudio, o qual pode ter uma ou mais de uma fonte, representada pelo atributo "src" ou pelo elemento <source>, também pode ser usado como mediastream.

  • <canvas>

    • Espaço usado para desenhar gráficos e animações.

  • <datalist>

    • Série de opções para o usuário escolher e o sistema registrar. Pesquisar na MDN o uso, pois envolve label e input.

  • <details>

    • Criará um ícone que mostra detalhes ao clicar. Deve incluir um elemento <summary> com os detalhes de fato.

  • <embed>

    • Incorpora conteúdo externo, como de um plug-in do browser (ex. vídeos). Está em desuso em alguns browsers, por segurança.

  • <ul>

    • Lista desordenada, geralmente com ícones.

  • <ol>

    • Lista em ordem.

  • <li>

    • Item em uma lista.

  • <select>
