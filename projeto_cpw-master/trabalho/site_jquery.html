Detalhes:

Principal finalidade:

Facilitar o uso do JavaScript e suas funcionalidades, simplificando muitas tarefas complexas como calls do AJAX, animações e manipulação de DOM.

O jQuery pega muitas tarefas comuns que requerem muitas linhas de código JavaScript para serem realizadas e as agrupa em métodos que você pode chamar com uma única linha de código. Por exemplo, no exemplo apresentado neste site, o jQuery facilita a manipulação de eventos e operações a partir do click nos botões da calculadora.

Principais funcionalidades:

  • Manipulação de CSS
  • Métodos de evento HTML
  • Efeitos/Animações
  • AJAX
  • Utilidades

Pré-requisitos:

Para utilizar o Jquery temos 2 opções:

Baixar a biblioteca do site jquery.com

Incluir o jQuery no documento a partir de uma rede de distribuição de conteúdo, como o google

Para fazer uma melhor utilização do jquery é necessário conhecimento previo de HTML/CSS e JavaScript
